Project Title
 Budget Tracker – Personal Finance Management System

📌 Problem Statement

Many students and working professionals find it difficult to manage their daily income and expenses. They often spend money without tracking it, making it hard to save or achieve financial goals. Existing finance apps are sometimes complex or require subscriptions, making them less suitable for beginners.

📖 Project Description

The Smart Budget Tracker is a simple web application that helps users manage their personal finances. Users can add their income and expenses, view their remaining balance, set a savings goal, and monitor their financial progress. The application also displays a pie chart to visualize income and expenses, making financial information easy to understand.

🎯 Objectives
Help users track their income and expenses.
Encourage better saving habits.
Provide a simple and user-friendly interface.
Visualize financial data with charts.
Enable users to monitor their savings progress.
🚀 Features
✅ Add Income
✅ Add Expenses
✅ Automatic Balance Calculation
✅ Savings Goal Tracker
✅ Progress Bar
✅ Pie Chart Visualization
✅ Transaction History
✅ Local Storage (Data remains after refresh)
⭐ How is this project different?

Unlike many existing budgeting applications, our project focuses on simplicity and accessibility.

Our unique features include:
Beginner-friendly interface.
Real-time balance calculation.
Savings goal with a visual progress bar.
Interactive pie chart for quick financial insights.
No login required.
Works directly in a web browser without installation.
Stores data locally, allowing users to continue even after refreshing the page.

🔥 Major Challenges Faced During Development

During the development of this project, we encountered several challenges:

Updating the balance automatically
Ensuring the balance recalculates immediately after every income or expense entry.
Synchronizing the pie chart
Keeping the chart updated whenever transaction data changes.
Managing local storage
Saving and retrieving data correctly after page refresh.
Responsive design
Making the website look good on both desktop and mobile devices.
